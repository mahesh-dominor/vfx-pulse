# Release Notes v1.0.0

## Highlights
- Authentication module with role-aware access.
- Dashboard module with KPI and activity data.
- Users, projects, sequences, shots, daily updates modules.
- Reviews, assets, reports, notifications, settings modules.
- Deployment baseline: Docker + CI/CD workflow.

## Technical
- Prisma schema expanded with soft-delete and audit fields.
- Service-layer business logic across modules.
- Zod validation at API boundaries.

## Known limitations
- Email delivery provider integration is placeholder.
- CSV/Excel/PDF exports are pending provider integration.
